动态REM 手机专用的自适应方案

首先了解常用的 CSS长度单位（详情查阅 length CSS mdn）：

  1. px像素
  2. pt
  3. em（一个M的宽度或者一个汉字的宽度）
  4. ex(一个X的高度）
  5. vh(viewport 100vh=视口高度）
  6. vw（100vw=视口宽度）
  7. rem (root em)根元素的font-size

我们可以在html用script写上这句话

意思是用js动态的让屏幕大小=rem 这样我们就可以设元素用百分比来保持布局了
